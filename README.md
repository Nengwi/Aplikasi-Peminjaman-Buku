## 🌟 Tentang Proyek
**PERPUSPRO** adalah sistem manajemen perpustakaan digital yang menggabungkan kekuatan **Laravel 11** di backend dan **React.js** di frontend melalui **Inertia.js**. Dibangun dengan estetika *futuristic dark mode*, aplikasi ini memberikan pengalaman pengelolaan data buku yang cepat, intuitif, dan responsif.

## 🚀 Fitur Unggulan

### 🔐 Multi-role Authentication & Authorization
Sistem membedakan akses antara **Administrator** dan **Siswa** secara ketat untuk menjaga integritas data.
* **Admin**: Akses penuh ke dashboard, kelola buku, transaksi, pengguna, dan laporan.
* **Siswa**: Akses katalog buku dan melihat riwayat peminjaman pribadi secara real-time.

### 📖 Manajemen Koleksi Buku (CRUD)
Pengelolaan data buku yang komprehensif tanpa perlu reload halaman.
* Input data: Judul, Penulis, Penerbit, Tahun, Stok, dan Lokasi Rak.
* Fitur Pencarian & Filter: Mencari buku berdasarkan judul atau kategori secara instan.

### 📋 Sistem Transaksi Terintegrasi
* Proses Peminjaman & Pengembalian yang otomatis memotong/menambah stok buku secara akurat.
* Validasi otomatis untuk mencegah peminjaman ganda atau stok kosong.

### 📊 Dashboard & Reporting
* Statistik visual untuk memantau aktivitas perpustakaan di Dashboard.
* Fitur cetak laporan periodik untuk arsip perpustakaan.

## 🎨 Keunggulan Desain (UI/UX)
* **Modern Dark Mode**: Menggunakan skema warna Slate & Blue yang nyaman di mata.
* **Glassmorphism**: Efek transparan pada navbar dan card untuk kesan premium.
* **Fully Responsive**: Tampilan tetap sempurna baik di desktop maupun perangkat mobile.

## 🛠️ Stack Teknologi
| Teknologi | Kegunaan |
| :--- | :--- |
| **Laravel 11** | Robust Backend Framework & API |
| **Inertia.js** | Menghubungkan Laravel & React tanpa API kompleks |
| **React.js** | Library Frontend untuk UI yang reaktif |
| **Tailwind CSS** | Utility-first CSS untuk styling modern |
| **Lucide React** | Paket ikon minimalis dan elegan |
| **MySQL** | Database Relasional untuk penyimpanan data aman |
