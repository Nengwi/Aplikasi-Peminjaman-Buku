*Fitur Utama*

🛡️ Multi-role Authentication: Akses berbeda antara Administrator untuk manajemen data dan Siswa untuk katalog & riwayat peminjaman.

📖 Management Buku: Operasi CRUD (Create, Read, Update, Delete) data buku lengkap dengan kategori, lokasi rak, dan status stok.

📑 Transaksi Peminjaman: Sistem peminjaman dan pengembalian buku yang terintegrasi.

📊 Laporan Real-time: Dashboard statistik dan fitur cetak laporan untuk memudahkan pemantauan aktivitas perpustakaan.

🎨 UI/UX Modern: Desain Dark Mode futuristik menggunakan Tailwind CSS dengan efek Glassmorphism dan Backdrop Blur.

⚡ Reactive Interface: Pengalaman navigasi cepat tanpa reload halaman berkat integrasi Inertia.js dan React.

🚀 Teknologi yang Digunakan
Aplikasi ini dibangun menggunakan stack teknologi modern:

Framework Backend: Laravel 11

Frontend Bridge: Inertia.js

Library Frontend: React.js

Styling: Tailwind CSS

Icons: Lucide React

Database: MySQL / MariaDB
